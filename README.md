# OSC Test Chamber

This repository provides design files for a chamber used to seal and measure organic solar cell devices.
The chamber can be fabricated using a 3D printer and commercially available components.

## Features

- Devices sealed under an inert atmosphere (e.g., glovebox) can be measured under ambient conditions  
- Multiple small devices can be encapsulated simultaneously  
- Sufficient airtightness for evaluation of organic solar cells  
- Literature-level OSC performance can be reproduced (in the author's environment)  
- Heating is not supported (limited by the thermal resistance of typical 3D-printing filaments)  

## Contents

This repository includes:
- STL files for 3D-printed parts  
- Parts list (BOM)  
- Example photos of the chamber
